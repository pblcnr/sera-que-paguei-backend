# Será que Paguei? - Backend

## 📊 Sobre o Projeto
Será que Paguei? veio para te ajudar a nunca mais esquecer de pagar suas contas. Cadastre suas contas e receba lembretes na data de vencimento.

## 🚀 Tecnologias
- Java 21
- Spring Boot
- PostgreSQL
- Docker

## 🏁 Como rodar o projeto

### Pré-requisitos
- Java 21
